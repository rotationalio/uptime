# Uptime

[![Tests](https://github.com/rotationalio/uptime/actions/workflows/tests.yaml/badge.svg)](https://github.com/rotationalio/uptime/actions/workflows/tests.yaml)

**Service status monitor for Rotational applications and systems.**

Uptime performs status checks on Rotational services on a periodic basis in order to detect service availability, currently deployed version, and to quickly report if any Rotational services are down or unavailable.
